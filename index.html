Housing In Melbourne: Beyond The Price Tag [Python]

This machine learning project predicts house prices in Melbourne using factors such as suburb, building area, latitude, and longitude. To achieve this, data pipelines were created for cleaning, feature encoding, and outlier management, successfully reducing the Mean Absolute Error (MAE) from $276K to $159K. After developing and evaluating four different models, Ridge Regression was selected as the optimized final model. The solution was then deployed as a Flask API on Render and as an interactive Jupyter Widget on Google Colab, allowing for real-time predictions.
